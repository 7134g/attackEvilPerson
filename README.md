# attackEvilPerson
Punish the vile villains

## 功能

- **URL 采集** (`collect`)：从搜索结果中正则匹配广告落地页 URL，支持并发采集、代理、去重
- **留言发送** (`send`)：通过浏览器自动化打开 URL，随机组合留言并发送
- **定时运行** (`cron`)：每天早上 9:00 自动执行留言发送

## 项目结构

```
attackEvilPerson/
├── main.go                        # 入口，子命令路由
├── config.yaml                    # 配置文件
├── internal/
│   ├── config/config.go           # 配置加载
│   ├── collector/collector.go     # URL 采集模块
│   ├── sender/sender.go           # 浏览器发送模块
│   └── message/builder.go         # 留言模板随机组合
├── data/
│   ├── kw_city.txt                # 城市关键词列表
│   └── kw_hospital.txt            # 医院关键词列表
└── api.txt                        # 采集输出的 URL 列表
```

## 使用方法

### 编译

```bash
go build .
```

### 配置

编辑 `config.yaml`，填写以下必填项：

- `tel_number`：目标手机号码
- `tel_name`：目标姓名
- `proxy`：HTTP 代理地址（可选，如 `http://127.0.0.1:7890`）
- `browser_path`：Chrome 浏览器路径（可选，Rod 可自动发现）

### 运行

```bash
# 采集 URL（百度搜索 -> api.txt）
attackEvilPerson collect

# 发送留言（读取 api.txt 逐个打开并留言）
attackEvilPerson send

# 定时模式（每天 9:00 自动执行 send）
attackEvilPerson cron
```
